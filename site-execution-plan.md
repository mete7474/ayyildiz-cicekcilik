# Ayyıldız Çiçekçilik Site Planı

## 1. Mevcut Durum
- Tek sayfa HTML prototipi
- Ürün katalogu, sepet, favoriler, arama, filtreleme, modal içerikler var
- Gerçek backend/sipariş/ödeme mekanizması yok

## 2. Eksikler
### 2.1. Altyapı
- Ürün veritabanı
- Sipariş yönetimi
- Kullanıcı / oturum açma / üyelik
- Ödeme entegrasyonu
- Stok ve envanter kontrolü

### 2.2. Kullanıcı Deneyimi
- Checkout formu yok
- Sipariş onayı / takip yok
- Ürün detay sayfaları yok
- Gerçek iletişim formları yok
- Mobil erişilebilirlik ve klavye desteği eksik

### 2.3. SEO & Hosting
- Meta tag ve Open Graph eksik
- Favicon / manifest yok
- Site haritası / robots.txt yok
- Alan adı + SSL gerekecek

## 3. Önerilen Adımlar
1. Frontend prototipi tamamla
2. Backend API tasarla
3. Veritabanı şeması oluştur
4. Checkout ve ödeme akışını oluştur
5. Admin paneli planla
6. SEO ve yayınlama adımlarını belirle

## 4. Öncelik Sıralaması
1. Checkout + sipariş altyapısı
2. Kullanıcı hesabı / oturum
3. Ürün detay sayfaları
4. Backend servis + API
5. Hosting + Alan adı + HTTPS
6. SEO, analytics, performans optimize
